## 🎯 액션별 처리 로직

### 1. BUY (일반 매수)

```python
신뢰도: 60% 이상
매수량: sim.buy_amount (기본 1,000,000원)
조건: AI 승인 + 스크리닝 통과
```

### 2. ADD_BUY (추가 매수) 🆕

```python
신뢰도: 70% 이상 (또는 BUY + 85% 이상)
매수량: sim.buy_amount x 배수
  - 신뢰도 90% 이상 → 3.0배
  - 신뢰도 80-90% → 2.5배
  - 신뢰도 70-80% → 2.0배
조건: 
  - 이미 보유 중인 종목
  - AI 추가 매수 권장
  - 포지션 한도 초과 허용
```

### 3. SELL (일반 매도)

```python
신뢰도: 60% 이상
우선순위: 2순위 (기술적 지표 다음)
처리:
  - AI가 SELL 권장
  - 기술적 지표 참고 (손절, 추적 손절)
```

### 4. STRONG_SELL (강력 매도) 🆕

```python
신뢰도: 50% 이상 (일반 SELL보다 낮음)
우선순위: 1순위 (최우선)
처리:
  - AI가 STRONG_SELL 권장
  - 기술적 지표 무시
  - 즉시 매도 실행
```
